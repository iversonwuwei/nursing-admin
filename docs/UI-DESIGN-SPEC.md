# 养老院管理系统 UI 设计规范
> 版本：v1.0.0 | 更新日期：2026-03-27
> 设计师：小画 | 技术栈：Next.js 14 + TypeScript + Tailwind CSS + shadcn/ui

---

## 1. 色彩系统

### 1.1 设计理念
**"温馨专业"** — 医疗/养老行业的设计需要在信任感与亲和力之间取得平衡：
- 避免过于冰冷的纯医疗蓝
- 避免过于幼稚的卡通风格
- 营造安全、舒适、被尊重的视觉感受

### 1.2 色彩语义

| 语义 | 色值 | 用途 |
|------|------|------|
| **主色-暖青** | `#0D9488` | 主要按钮、导航高亮、链接 |
| **主色浅** | `#CCFBF1` | 背景色块、标签底色 |
| **辅助-暖橙** | `#F97316` | 重要操作、老人头像/标签 |
| **辅助-暖金** | `#D97706` | 次要强调、徽章 |
| **背景-米白** | `#FAFAF8` | 页面底色 |
| **卡片-纯白** | `#FFFFFF` | 卡片、弹层背景 |
| **边框-浅灰** | `#E7E5E4` | 分割线、输入框边框 |
| **文本-深灰** | `#1C1917` | 标题、一级文字 |
| **文本-中灰** | `#57534E` | 正文、二级文字 |
| **文本-浅灰** | `#A8A29E` | 占位符、三级文字 |

### 1.3 功能色

| 功能 | 色值 | 说明 |
|------|------|------|
| **成功** | `#059669` | 操作成功、正常状态 |
| **警告** | `#D97706` | 需关注、待处理 |
| **危险** | `#DC2626` | 紧急、健康异常 |
| **信息** | `#0284C7` | 提示信息、说明文字 |

### 1.4 告警色梯度

```
紧急 → #B91C1C (深红)
严重 → #DC2626 (红)
警告 → #F59E0B (橙)
注意 → #FCD34D (浅黄)
```

## 2. 字体系统

### 2.1 中文字体

| 用途 | 字体 | 备选 |
|------|------|------|
| **界面正文** | "PingFang SC", "Microsoft YaHei" | 系统默认 |
| **标题强调** | "PingFang SC Medium" | "Microsoft YaHei" |

> macOS 用户使用 PingFang SC，Windows 用户使用 Microsoft YaHei，保证各平台最佳渲染。

### 2.2 英文字体

| 用途 | 字体 |
|------|------|
| **界面正文** | Inter |
| **数字/数据** | JetBrains Mono |

### 2.3 字重规范

| 字重 | 数值 | 用途 |
|------|------|------|
| Regular | 400 | 正文内容 |
| Medium | 500 | 按钮文字、标签 |
| Semibold | 600 | 卡片标题、导航项 |
| Bold | 700 | 页面大标题 |

### 2.4 字号规范

```
xs   → 12px / 0.75rem  （辅助说明、时间戳）
sm   → 14px / 0.875rem  （表格内容、标签）
base → 16px / 1rem      （正文）
lg   → 18px / 1.125rem （小标题）
xl   → 20px / 1.25rem  （卡片标题）
2xl  → 24px / 1.5rem    （页面子标题）
3xl  → 30px / 1.875rem （页面大标题）
```

### 2.5 行高规范

| 场景 | 行高 |
|------|------|
| 标题 | 1.2–1.3 |
| 正文 | 1.5–1.6 |
| 紧凑列表 | 1.3 |

## 4. 圆角与阴影

### 4.1 圆角规范

| 组件 | 圆角 | Tailwind class |
|------|------|----------------|
| 页面/大区块 | 16px | `rounded-2xl` |
| 卡片 | 12px | `rounded-xl` |
| 按钮 | 8px | `rounded-lg` |
| 输入框 | 8px | `rounded-lg` |
| 标签/徽章 | 6px | `rounded-md` |
| 小图标按钮 | 8px | `rounded-lg` |
| 头像 | 全圆 | `rounded-full` |

### 4.2 阴影规范

| 等级 | 用途 | CSS |
|------|------|-----|
| **sm** | 列表项 hover | `0 1px 2px 0 rgb(0 0 0 / 0.05)` |
| **md** | 卡片默认 | `0 4px 6px -1px rgb(0 0 0 / 0.07), 0 2px 4px -2px rgb(0 0 0 / 0.05)` |
| **lg** | 弹层/下拉 | `0 10px 15px -3px rgb(0 0 0 / 0.08), 0 4px 6px -4px rgb(0 0 0 / 0.05)` |
| **xl** | 模态框 | `0 20px 25px -5px rgb(0 0 0 / 0.1), 0 8px 10px -6px rgb(0 0 0 / 0.05)` |

### 4.3 交互状态阴影

```css
/* 按钮 hover */
hover:shadow-md

/* 卡片 hover */
hover:shadow-lg
transition-shadow duration-200

/* 模态框 */
animate-in fade-in zoom-in-95
```

## 5. 组件设计规范

### 5.1 按钮

#### 主按钮
```css
背景: #0D9488 (teal-600)
文字: #FFFFFF
hover: #0F766E (teal-700)
圆角: rounded-lg (8px)

```

#### 次按钮
```css
背景: #FFFFFF
边框: 1px solid #E7E5E4
文字: #1C1917
hover: 背景 #FAFAF8
圆角: rounded-lg
```

#### 危险按钮
```css
背景: #DC2626 (red-600)
文字: #FFFFFF

```

#### 幽灵按钮
```css
背景: transparent
文字: #0D9488
hover: 背景 #CCFBF1 (teal-100)
```

#### 按钮尺寸

| 尺寸 | 高度 | padding | 字号 |
|------|------|---------|------|
| sm | 32px | 8px 12px | 14px |
| md (默认) | 40px | 10px 16px | 14px |
| lg | 48px | 12px 24px | 16px |
| icon | 36px × 36px | — | — |

#### 按钮状态
- **Default** — 标准色
- **Hover** — 颜色加深 + shadow-md
- **Active** — 颜色更深，scale(0.98)
- **Disabled** — opacity 0.5, cursor-not-allowed
- **Loading** — 显示 spinner + 文字"处理中…"

## 7. 动效规范

### 7.1 过渡时长

| 类型 | 时长 | 说明 |
|------|------|------|
| 微交互 | 150ms | 按钮hover, 颜色变化 |
| 展开/收起 | 200ms | 下拉菜单, 折叠面板 |
| 页面元素 | 300ms | 卡片入场, 模态框 |
| 整体页面 | 400ms | 路由切换, 大区块 |

### 7.2 缓动函数

```css
/* 默认 */
transition: all 200ms ease-out

/* 弹性 (弹窗/下拉) */
transition: all 300ms cubic-bezier(0.4, 0, 0.2, 1)

/* 入场动画 */
animate-in fade-in slide-in-from-bottom-4 duration-300
```

### 7.3 加载状态

- **骨架屏** (Skeleton): bg-gray-200, rounded, shimmer动画
- **Spinner**: teal-600, 20px
- **进度条**: teal-500, rounded-full

## 8. 辅助规范

### 8.1 图标规范

- **图标库**: Lucide React (与 shadcn/ui 一致)
- **默认尺寸**: 20px (界面), 16px (紧凑区域), 24px (大标题旁)
- **颜色**: inherit (跟随文字颜色) 或 gray-500

### 8.2 空状态

│                             │

```

内容: 清晰说明操作后果
按钮: [取消] [确认操作] (危险色)
```
